# Executive Summary

**Product:** SubSense (Subscription Intelligence Platform)

**One-liner:** Never get charged unknowingly. Detect, track, and optimize recurring payments.

**Key Differentiators:**
- India-first SMS/email parsing
- Event-driven reminders with strong consistency
- Actionable insights (waste detection, cancel recommendations)

**Tech Highlights:**
- Modular monolith → microservices ready
- PostgreSQL + Redis + Queue (BullMQ/Kafka-ready)
- Observability (logs, metrics, tracing)

**Business Impact:**
- Reduce wasted spend
- Increase financial awareness
